# 将棋ゲーム 使い方ガイド

お父様へ：この将棋ソフトの使い方を説明します。

---

## 起動方法

### デスクトップから起動

デスクトップにある「将棋ゲーム」のアイコンをダブルクリックしてください。

### スタートメニューから起動

1. 画面左下のWindowsマーク（スタートボタン）をクリック
2. 「将棋ゲーム」と入力して検索
3. 表示された「将棋ゲーム」をクリック

---

## ゲームの始め方

### 1. モード選択

ソフトを起動すると、最初にモードを選択する画面が表示されます。

- **通常将棋**: コンピュータと対局します
- **詰将棋**: 詰将棋の問題を解きます

遊びたいモードをクリックしてください。

### 2. 難易度選択

次に難易度を選択します。

**通常将棋の場合:**
- **初級**: 弱いコンピュータ（初心者向け）
- **中級**: 少し強いコンピュータ
- **上級**: 強いコンピュータ
- **超上級**: とても強いコンピュータ（上級者向け）

**詰将棋の場合:**
- **初級**: 1手で詰む問題
- **中級**: 3手で詰む問題
- **上級**: 5手で詰む問題
- **超上級**: 7手以上で詰む問題

### 3. ゲーム開始

「ゲームを開始しますか？」と表示されます。

- **はい**: ゲームが始まります
- **いいえ**: 難易度選択画面に戻ります

---

## 駒の動かし方

### 盤上の駒を動かす

1. **動かしたい駒をクリック**
   - 駒が青い枠で囲まれます
   - 動かせるマスが緑色で表示されます

2. **移動先のマスをクリック**
   - 駒が移動します
   - 成れる場合は「成りますか？」と聞かれます

### 持ち駒を打つ

1. **画面右側の持ち駒エリアの駒をクリック**
   - 持ち駒が青い枠で囲まれます
   - 打てるマスが緑色で表示されます

2. **打ちたいマスをクリック**
   - 駒が打たれます

### 選択を取り消す

- 別の場所をクリックすると、選択が解除されます

---

## メニューの使い方

画面上部にメニューがあります。

### ファイルメニュー

- **新規ゲーム**: 新しいゲームを始めます
- **保存**: 現在のゲームを保存します
- **読み込み**: 保存したゲームを再開します
- **終了**: ソフトを終了します

### 設定メニュー

- **UIカスタマイズ**: 画面の見た目を変更できます
  - 文字の大きさ
  - 盤の色
  - 駒の画像
  - 音のオン/オフ

## 画面の見方

```
┌─────────────────────────────────────────┐
│ ファイル  設定  ヘルプ                   │  ← メニューバー
├─────────────────────┬───────────────────┤
│                     │  後手の持ち駒      │
│                     │  ┌─────┐         │
│                     │  │ 歩 ×3│         │
│      将棋盤         │  │ 角   │         │
│   （9×9のマス）     │  └─────┘         │
│                     │                   │
│                     │  先手の持ち駒      │
│                     │  ┌─────┐         │
│                     │  │ 銀   │         │
│                     │  │ 桂 ×2│         │
│                     │  └─────┘         │
└─────────────────────┴───────────────────┘
```

### 盤面の記号

- **段**: 縦の番号（一、二、三...九）
- **筋**: 横の番号（1、2、3...9）
- **後手の駒**: 上向き（逆さま）
- **先手の駒**: 下向き（正常）

### ハイライト

- **青い枠**: 選択中の駒
- **緑の印**: 動かせる/打てるマス
- **オレンジの枠**: 直前に動いた駒

---

## ゲームの終わり方

### 勝敗

- **詰み**: 相手の王を詰ませると勝ちです
- **投了**: メニューから投了できます
- **引き分け**: 千日手（同じ局面が繰り返される）は引き分けです

### 次のゲーム

1. メニューから「ファイル」→「新規ゲーム」をクリック
2. またはソフトを再起動

---

## 保存と再開

### ゲームの保存

1. メニューから「ファイル」→「保存」をクリック
2. ファイル名を入力（例：「2026年4月の対局」）
3. 「保存」をクリック

### 保存したゲームの再開

1. メニューから「ファイル」→「読み込み」をクリック
2. 再開したいファイルを選択
3. 「開く」をクリック

**注意**: 保存したゲームは、盤面と持ち駒がそのまま再現されます。

---

## トラブルシューティング

### ソフトが起動しない

1. パソコンを再起動してみてください
2. もう一度アイコンをダブルクリックしてみてください

### 駒が動かせない

- **相手の番かもしれません**: 画面をよく見て、自分の番か確認してください
- **ルール違反**: その駒はそこに動かせません
  - 二歩（同じ筋に歩が2つ）
  - 王手放置（王が取られる状態）
  - などのルール違反があります

### 画面が小さい/大きい

メニューから「設定」→「UIカスタマイズ」→「文字サイズ」で調整できます。

### 音が鳴らない

メニューから「設定」→「UIカスタマイズ」→「音声」をオンにしてください。

## よくある質問

### Q: コンピュータはどのくらい強いですか？

**A:** 
- **初級**: 初心者の方向け
- **中級**: ある程度駒の動きを覚えた方向け
- **上級**: 将棋に慣れた方向け
- **超上級**: かなり強いです

### Q: 詰将棋の答えがわからない

**A:** 「ファイル」→「新規ゲーム」で別の問題に変えられます。

### Q: ゲームの途中で休憩したい

**A:** 「ファイル」→「保存」で保存して、ソフトを終了できます。
次に起動した時に「読み込み」で続きから再開できます。

### Q: 対局の記録を見たい

**A:** 現在の版では対局記録の表示機能はありません。
必要であれば、紙にメモを取ることをお勧めします。

## 将棋のルールについて

このソフトは正式な将棋のルールに従っています：

- **二歩禁止**: 同じ筋に自分の歩を2枚置けません
- **王手放置禁止**: 王が取られる状態のままにできません
- **打ち歩詰め禁止**: 歩を打って相手を詰ますことはできません
- **行き所のない駒**: 動けなくなる場所に駒を置けません
- **千日手**: 同じ局面が繰り返されると引き分けです
